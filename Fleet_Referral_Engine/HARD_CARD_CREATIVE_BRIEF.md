# HARD CARD CREATIVE BRIEF
## SpiceX Fleet-to-Pipeline Referral Engine
## Handoff: Graphic Design Team
## Date: April 2026

---

> **OBJECTIVE**: Design a premium, VIP-style physical card that bridges SpiceX's fleet encounters (truck/Tesla) to digital conversion. This card should feel like receiving an invitation to an exclusive program — not a flyer.

---

## PHYSICAL SPECIFICATIONS

| Attribute | Spec |
|---|---|
| **Dimensions** | 8.5" × 5.5" (half-letter, landscape orientation) |
| **Stock** | 32pt C2S (Coated Two Sides) |
| **Lamination** | Soft-Touch Matte (both sides) |
| **Special Finishes** | Spot UV on SpiceX logo + QR code area. Metallic Red foil stamp on the "X" in SpiceX. |
| **Corners** | 0.25" radius rounded |
| **Edge Treatment** | Painted edge — SpiceX Red (#D6001C) |
| **Color Profile** | CMYK (design in CMYK, NOT RGB) |
| **Resolution** | All assets at 300 DPI minimum |
| **Bleed** | 0.125" bleed on all sides |
| **Safe Zone** | 0.25" inset from trim on all sides |
| **Quantity (Initial Run)** | 500 units |

### Detailed Spec

**Left Column (60% of card width)**

- Headline: `KNOW SOMEONE WHO NEEDS THIS?`
  - Font: Montserrat ExtraBold, 22pt, White
  - Line break after "SOMEONE"

- Subhead: `EARN $500 FOR EVERY QUALIFIED REFERRAL.`
  - Font: Oswald Regular, 15pt, SpiceX Red (#D6001C)

- Three Steps (vertical stack with left-aligned numbers):
  - `① SCAN THE CODE`
  - `② SUBMIT YOUR REFERRAL (60 SECONDS)`
  - `③ GET PAID WHEN THEY CLOSE`
  - Numbers: Rendered as filled SpiceX Red circles with white numerals inside (18px diameter)
  - Text: Montserrat SemiBold, 10pt, White
  - Spacing: 16px between each step

- Micro-text: `No cap on referrals. No limit on earnings.`
  - Font: Oswald Regular, 7pt, #B0B8C1, italic

**Right Column (40% of card width)**

- QR Code: 2.5" × 2.5" minimum

  - SpiceX "X" logo embedded in QR center (part of the code pattern, not overlaid)
  - Apply Spot UV to the entire QR code area (makes it pop with subtle gloss)
  - URL encoded: `https://spicex.com/go?src=truck_hardcard&batch=FLEET_Q2_2026`

- Below QR: `spicex.com/go`
  - Font: Space Grotesk Bold, 12pt, White
  - This is the fallback memorizable URL

- Below URL: Lock icon + `Secure · Private · Fast`
  - Font: Oswald Regular, 6pt, #B0B8C1

**Background Treatment**
- #050505 black base
- Right 40% has a dramatic gradient wash: SpiceX Red at 15% opacity, fading to pure black
- This creates visual "energy" drawing the eye toward the QR code

**Bottom Edge (Full Width)**
- Red divider line: 0.5pt, same as Side A
- `Mike Ryan · CEO, SpiceX · spicex.com`
- Font: Montserrat Medium, 6pt, #B0B8C1

## QR CODE GENERATION NOTES

- Generate the QR code using a high-quality generator that supports logo embedding
- Recommended: QR Code Monkey (qrcode-monkey.com) or Adobe Express
- Error correction level: H (High — 30% recovery, needed for logo embedding)
- Module color: White (#FFFFFF)
- Background: Transparent (printed on dark card)
- Minimum quiet zone: 4 modules on all sides
- **TEST THE QR CODE AT PRINT SIZE before final production run**
